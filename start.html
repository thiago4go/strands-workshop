Introducing Strands

An Open Source SDK for Building AI Agents

From the AWS Open Source Blog

What is Strands?

Strands is an open-source Python SDK designed to simplify the creation of AI agents. It provides a structured, event-driven framework that helps developers build, test, and scale sophisticated agent-based applications. By handling the complexities of state management, concurrency, and tool integration, Strands lets you focus on the agent's logic and capabilities.

Key Features

Stateful & Concurrent

Agents in Strands are stateful by default, remembering context across multiple interactions. The framework manages concurrency, allowing agents to handle multiple tasks simultaneously.

Event-Driven Architecture

Built on an event bus, Strands allows for loose coupling between components. Agents react to events, making the system modular, extensible, and easy to debug.

Multi-Agent Collaboration

Orchestrate multiple agents for complex tasks using patterns like workflows, graphs, or swarms. Strands models collaboration as a tool, allowing the primary agent to delegate tasks.

Pluggable Models & Tools

Easily integrate various LLMs (like Amazon Bedrock, OpenAI, etc.) and custom tools. The SDK provides a consistent interface for tool definition and model interaction.

Observability

With its event-driven nature, Strands makes it easy to trace the execution of an agent. You can log every event and state change, providing clear insights into the agent's behavior.

Local-First Development

Develop and test your agents entirely on your local machine. Strands comes with an in-memory event bus and tools for a smooth local development experience.

Example: Crafting a Prompt

Strands constructs a detailed prompt for the LLM by combining the agent's persona, available tools, and conversation history (memory).

How It Works: The Agentic Loop

1

Run

The agent is invoked with a user's input, which is added to its memory.

2

Think

The agent uses an LLM to decide the next step: either respond to the user or use a tool.

3

Act

If a tool is chosen, the agent executes it, records the result in memory, and the loop continues.

Get Started with Strands

Ready to build your own AI agents? Strands is open source and available on GitHub. You can install it via pip and start building in minutes. 

$ pip install strands-agents
